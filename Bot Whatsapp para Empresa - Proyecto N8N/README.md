# WhatsApp Bot Automation with n8n

This project is a simple WhatsApp automation bot built with n8n.

## Description

The bot receives a message from a user, sends a numbered menu with options and stores the customer information automatically in an Excel spreadsheet.

## Technologies used

- n8n
- Webhooks
- Twilio API
- Excel / Google Sheets
- Workflow Automation

## Workflow

1. User sends a message on WhatsApp
2. Webhook receives the message
3. n8n processes the request
4. Bot responds with menu options
5. User selection is stored in Excel

## Files

- workflow.json → exported workflow from n8n
